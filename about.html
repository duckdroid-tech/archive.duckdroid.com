About this project

Duckdroid's Nostalgic Android Archives is a fan website of collection of Android-related artifacts: versions, imagery, themes and other nostalgic content. This site aims to present packages clearly and make them easy to browse across devices.

Designs

This Duckdroid's Android archive website is redesigned the Material Design 2 branding. Sorry, Material You!.

Contribute

If you'd like to contribute packages or images, please open an issue or a pull request on the GitHub repository.
